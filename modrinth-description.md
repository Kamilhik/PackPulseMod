# PackPulse

Keep a Fabric client pack in sync from a remote manifest.

PackPulse checks a JSON manifest when the game starts, shows a file list before download, then downloads missing or changed files and shows in-game sync progress.

## Good for

- private servers
- friend groups
- curated modpacks
- custom launcher-free distributions

## Notes

If `.jar` mods change, Minecraft closes so you can start it again with updated mods.
